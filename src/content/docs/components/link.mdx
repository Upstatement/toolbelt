## Usage

```html
<a x-link:current href="/components/link">Link to the current page </a>
<a x-link:external href="https://upstatement.com">Link to an external site</a>
```

## Demo

<Card>
  <div class="flex flex-col">
      <a x-init="" x-link:current="" href="/components/link">
        <span>Link to the current page</span>

  </div>

Inspect the links above. Notice that new attributes have been added to the `<a>` tags.

```html ins='aria-current="page"' ins='target="_blank" rel="noopener noreferrer"'
<a x-link:current="" href="/components/link" aria-current="page">
  <span>Link to the current page</span>
</a>

<a x-link:external="" href="https://upstatement.com" target="_blank" rel="noopener noreferrer">
  <span>Link to an external site</span>
</a>
```
