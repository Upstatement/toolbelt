## Anatomy

```html
<a href="#" x-link:current></a>
<a href="#" x-link:external></a>
```

## Directives

The `x-link` family of directives is different from other components. These directives are independent of each other and can be used in isolation.

<DirectiveReference directive="x-link:current" required>
  <Table>
    Checks whether the link points to the current page. This will automatically
    set `aria-current=page` on the link. By default, this will check that the link's
    [pathname](https://developer.mozilla.org/en-US/docs/Web/API/URL/pathname) matches that of the current page.
  </Table>

  <Table>

| Modifier  | Description                                                                    |
| --------- | ------------------------------------------------------------------------------ |
| `.nested` | If set, this directive will be applied to all `<a>` nested inside the element. |

  </Table>

  <Table>

| Data Attribute   | Values            |
| ---------------- | ----------------- |
| `[data-current]` | `true` \| `false` |

  </Table>
</DirectiveReference>

<DirectiveReference
  directive="x-link:external"
  required
>
  <Table>
    Checks whether the link points to an external website. This will automatically set the link to open in a new tab. By default, this will check whether the link shares the same origin as the current site.
  </Table>

  <Table>

| Data Attribute    | Values            |
| ----------------- | ----------------- |
| `[data-external]` | `true` \| `false` |

  </Table>

  <Table>

| Modifier  | Description                                                                    |
| --------- | ------------------------------------------------------------------------------ |
| `.nested` | If set, this directive will be applied to all `<a>` nested inside the element. |

  </Table>
</DirectiveReference>
