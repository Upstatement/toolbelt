## Anatomy

```html
<div x-flyout>
  <button x-flyout:trigger></button>
  <div x-flyout:content></div>
</div>
```

## Directives

<DirectiveReference
  directive="x-flyout"
  required
>
  The root element for the flyout.

  <Table>

| Modifier     | Description                                                                   |
| ------------ | ----------------------------------------------------------------------------- |
| `.hoverable` | Opens flyout on hover. Flyout automatically closes when cursor moves outside. |

  </Table>

  <Table>

| Data Attribute | Values             |
| -------------- | ------------------ |
| `[data-state]` | `open` \| `closed` |

  </Table>
</DirectiveReference>

<DirectiveReference
  directive="x-flyout:trigger"
  required
>
  Trigger button that opens or closes the flyout content.

  <Table>

| Data Attribute | Values             |
| -------------- | ------------------ |
| `[data-state]` | `open` \| `closed` |

  </Table>
</DirectiveReference>

<DirectiveReference
  directive="x-flyout:content"
  required
>
  The content that will be shown or hidden.

  <Table>

| Data Attribute | Values             |
| -------------- | ------------------ |
| `[data-state]` | `open` \| `closed` |

  </Table>
</DirectiveReference>
