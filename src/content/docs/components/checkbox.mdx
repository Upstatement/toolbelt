## Anatomy

```html
<div x-checkbox>
  <button x-checkbox:indicator></button>

</div>
```

## Directives

<DirectiveReference
  directive="x-checkbox"
  required
>
  The root element for the checkbox.

  <Table>

| Modifier   | Description                    |
| ---------- | ------------------------------ |
| `.checked` | Checks the checkbox by default |

  </Table>

  <Table>

| Event               | Description                                   | `detail`               |
| ------------------- | --------------------------------------------- | ---------------------- |
| `x-checkbox:change` | Whether the checkbox is checked or unchecked. | `{ checked: boolean }` |

  </Table>

  <Table>

| Data Attribute | Values                   |
| -------------- | ------------------------ |
| `[data-state]` | `checked` \| `unchecked` |

  </Table>
</DirectiveReference>
