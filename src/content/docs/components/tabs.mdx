## Anatomy

```html
<div x-tabs>
  <div x-tabs:list>
    <button x-tabs:tab></button>
  </div>

  <div x-tabs:panel></div>
</div>
```

## Directives

<DirectiveReference
  directive="x-tabs"
  required
>
  The root element for tabs.

  <Table>

| Value                                                              |
| ------------------------------------------------------------------ |
| The default tab to be displayed. Otherwise, display the first tab. |

  </Table>

  <Table>

| Event           | Description            | `detail`          |
| --------------- | ---------------------- | ----------------- |
| `x-tabs:change` | The tab that is opened | `{ tab: string }` |

  </Table>

</DirectiveReference>

<DirectiveReference
  directive="x-tabs:list"
  required
>
  List of tabs.

  <Table>

| Modifier                     | Description                                                                      |
| ---------------------------- | -------------------------------------------------------------------------------- |
| `.loop`                      | Loops keyboard navigation (e.g. arrow keys)                                      |
| `.automatic`                 | Automatically show/hide panel with keyboard navigation (e.g. arrow keys)         |
| `.vertical` \| `.horizontal` | Orientation of the accordion for keyboard navigation. Defaults to `.horizontal`. |

  </Table>

  <Table>

| Event           | Description            | `detail`          |
| --------------- | ---------------------- | ----------------- |
| `x-tabs:change` | The tab that is opened | `{ tab: string }` |

  </Table>
</DirectiveReference>

<DirectiveReference
  directive="x-tabs:tab"
  required
>
  Trigger button that shows the corresponding panel.

  <Table>

| Value                  |
| ---------------------- |
| The value for this tab |

  </Table>

  <Table>

| Event           | Description                       | `detail`            |
| --------------- | --------------------------------- | ------------------- |
| `x-tabs:change` | Whether the tab is open or closed | `{ open: boolean }` |

  </Table>

  <Table>

| Data Attribute | Values                 |
| -------------- | ---------------------- |
| `[data-state]` | `active` \| `inactive` |

  </Table>
</DirectiveReference>

<DirectiveReference
  directive="x-tabs:panel"
  required
>
  Panel that contains content.

  <Table>

| Value                                                         |
| ------------------------------------------------------------- |
| The value for this panel, which should match the correct tab. |

  </Table>

  <Table>

| Event           | Description                         | `detail`            |
| --------------- | ----------------------------------- | ------------------- |
| `x-tabs:change` | Whether the panel is open or closed | `{ open: boolean }` |

  </Table>

  <Table>

| Data Attribute | Values                 |
| -------------- | ---------------------- |
| `[data-state]` | `active` \| `inactive` |

  </Table>
</DirectiveReference>
